<h1>Hernani Samuel's Portfolio</h1>
<p>Welcome to the repository of my personal portfolio! This project represents not only my technical skills as a front-end developer but also my journey and growth in the field of web development. Here, you’ll find a collection of projects and skills that showcase my experience and passion for programming.</p>

<h2>About the Project</h2>
<p>This portfolio was built using a combination of modern front-end technologies, centered around the React ecosystem and the Flask framework for Python. The goal of this project is to provide a comprehensive overview of my technical skills, completed projects, and information about my career and background.</p>

<h2>Technologies Used</h2>
<ul>
  <li><strong>FastAPI:</strong> Used in some projects to demonstrate my back-end development skills.</li>
  <li><strong>React:</strong> To build an interactive and modern user interface.</li>
  <li><strong>CSS:</strong> For custom and responsive styling.</li>
  <li><strong>JavaScript:</strong> For front-end logic and interactivity.</li>
</ul>

<h2>Project Structure</h2>
<ul>
  <li><strong>src/components:</strong> Contains all React components used in the portfolio, including Banner, Skills, Projects, and more.</li>
  <li><strong>public:</strong> Stores static resources such as images and the résumé in PDF format.</li>
  <li><strong>src/assets:</strong> Contains images and icons used throughout the project.</li>
</ul>
